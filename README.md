this code is the finial version as of the end of video season 6 episode 10.  The code is intended to be used on an Arduino UNO.  
the code is experimental and I take no responsibility for issues.  The code was tested on a 212cc predator engine with and without a load.
the engine was tested at RPM range 1400 to 3600.
The load device that was used put a heavy load on the engine, but not enough of a load that would absorb the full 6.5 HP.
I offer the code for viewing, however I have no intent of maintaining this code or providing any technical support. 
the code is purely experimental and use at your own risk.

for the parts that were used in this project, check the video description of youtube channel Robot Cantina season 6 episode 10.
212efi 6x.stl is the 3d print file for the injector holder.  I used ASA plastic.
212 vac port3.stl is the vacuum port for the MAP sensor.  This part is intended to be threades with a 1/8 NPT tap
